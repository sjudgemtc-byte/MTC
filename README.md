# MTC

This repository is the stable systems layer for MTC work.

## Purpose

This repo is for reusable project assets, not day-to-day live teaching work.

Use it for:
- classroom systems
- reusable activity templates
- branded teaching collateral
- process documents
- role and onboarding frameworks
- safe generated notes and scripts

Do not use it as the main live workspace for:
- student-specific evidence
- daily working admin files
- sensitive records
- bulky output folders

## Working rule

- `iCloud / local working folders` = live daily work
- `GitHub` = stable systems, templates, and reusable project structure

See also: [GITHUB_WORKFLOW_RULE.md](/Users/simonjudge/Documents/MTC/GITHUB_WORKFLOW_RULE.md)

## Included areas

- `04_Classroom_Activities/`
- `07_Templates/`
- `templates/presentations/`
- selected safe files in `generated/`
- `AGENTS.md`

## Excluded areas

The Git setup deliberately ignores live or sensitive folders such as:
- `06_Student_Evidence/`
- `outputs/`
- most of `generated/`
- active admin and archive folders

## Notes

This setup is designed to stay workable over time:
- build in iCloud first
- promote reusable systems into GitHub
- avoid editing the same item in both environments
