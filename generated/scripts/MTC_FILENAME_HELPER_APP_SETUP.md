# MTC Filename Helper App Setup

## What This Is

This is a small desktop app for creating MTC audit-safe filenames.

Files:

- `mtc_filename_helper_app.py`
- `launch_mtc_filename_helper.command`

## Use On This Mac

1. Go to:
   `/Users/simonjudge/Documents/MTC/generated/scripts/`
2. Double-click:
   `launch_mtc_filename_helper.command`
3. Fill in the fields.
4. Click `Generate Filename`.
5. Click `Copy Filename` if needed.

## Move To The MacBook Air

Copy these files to the same folder on the MacBook Air:

- `mtc_filename_helper.py`
- `mtc_filename_helper_app.py`
- `launch_mtc_filename_helper.command`

Then open:

`launch_mtc_filename_helper.command`

## If macOS blocks the launcher

If macOS says the file is from an unidentified developer:

1. Right-click the `.command` file.
2. Choose `Open`.
3. Click `Open` again.

You should only need to do this once.

## Notes

- Student mode is for participant and assessment files.
- Resource mode is for worksheets, slides, templates, and trainer files.
- The app copies the generated filename to the clipboard.
