# MTC Filename Helper App Teaching Guide

## Purpose

Use this guide to teach another person how to use the MTC Filename Helper App.

The app helps create clear, audit-safe filenames for:

- student files
- worksheets
- trainer notes
- PowerPoint files
- classroom resources

## What The App Does

The app creates filenames in the MTC style.

Example:

`2026-06-07_MTC_Redfern_RED-01_Pathway_Williams-Mark_PA-Comments_BSBTEC203.docx`

## Before You Start

Make sure the person has access to:

- the MTC GitHub repo or copied app files
- a Mac
- the launcher file:
  `launch_mtc_filename_helper.command`

## Simple Way To Explain It

You can say:

`This app helps you make file names the right way. It saves time and keeps names clear and consistent.`

## How To Open The App

1. Go to the folder that contains the app files.
2. Double-click `launch_mtc_filename_helper.command`.
3. Wait for the app window to open.

If Mac security blocks it:

1. Right-click the file.
2. Choose `Open`.
3. Click `Open` again.

## What To Show First

Show the learner the two file types:

- `Student file`
- `Teaching resource or general file`

Explain:

- `Student file` is for participant work, evidence, assessment comments, or student records.
- `Teaching resource or general file` is for worksheets, slides, trainer notes, templates, or other teaching files.

## How To Use Student File Mode

1. Select `Student file`.
2. Enter the date.
3. Enter the document type.
4. Enter the unit or task if needed.
5. Enter the extension such as `docx` or `pdf`.
6. Enter the student surname.
7. Enter the student first name.
8. Click `Generate Filename`.
9. Click `Copy Filename` if needed.

## How To Use Resource Mode

1. Select `Teaching resource or general file`.
2. Enter the date.
3. Enter the document type.
4. Enter the unit or task if needed.
5. Enter the extension.
6. Enter the resource title.
7. Click `Generate Filename`.
8. Click `Copy Filename`.

## What To Practise

Give the learner 3 short practice tasks:

1. Create a filename for a student assessment comment file.
2. Create a filename for a worksheet.
3. Create a filename for a PowerPoint presentation.

## Good Teaching Script

You can say:

`First choose the file type. Then fill in the boxes. Then click Generate Filename. Check the result. If it looks correct, copy it and use it when you save the file.`

## Things To Check

Ask the learner:

- Did you choose the right file type?
- Did you enter the student name correctly?
- Did you use the right document type?
- Did you add the correct extension?
- Does the filename look clear and professional?

## Common Mistakes

- using the wrong file type
- forgetting the student name
- leaving out the file extension
- using a vague document type like `file` or `notes`
- not checking the final filename before copying

## Quick Examples

Student file:

`2026-06-07_MTC_Redfern_RED-01_Pathway_Williams-Mark_PA-Comments_BSBTEC203.docx`

Teaching resource:

`2026-06-07_MTC_Redfern_RED-01_Pathway_Canvas-Phone-Access_Student_Worksheet.docx`

## Trainer Tip

Teach this by modelling one example first, then letting the learner do the next two themselves.
