# Google Forms Computer Sign In / Sign Out Setup Instructions

Prepared: 2026-06-07

Use this guide for: Paul Martin or Chris Cook

Purpose: set up one simple Google Form that students can use every day to sign in and sign out of classroom computers.

## What you are creating

You are setting up:
- one Google Form
- one linked Google Sheet
- one form link that can be used every day

Students will use the same form:
- once when they start using a computer
- once when they finish

## Before you start

Make sure you have:
- a Google account signed in
- a list of the classroom computer numbers
- access to Google Forms
- access to Google Sheets

## Part 1: Create the form

1. Open Google Forms.
2. Click `Blank form`.
3. In the title, type:
   `Computer Sign In / Sign Out`
4. In the form description, type:
   `Please complete this form when you start and finish using a classroom computer.`

## Part 2: Add the questions

Add these 3 questions only.

### Question 1

1. Add a question.
2. Type:
   `Name`
3. Set the question type to:
   `Short answer`
4. Turn on:
   `Required`

### Question 2

1. Add another question.
2. Type:
   `Computer`
3. Set the question type to:
   `Dropdown`
4. Add the computer choices, for example:
   - `Computer 1`
   - `Computer 2`
   - `Computer 3`
   - `Computer 4`
   - `Computer 5`
5. Keep adding all classroom computers.
6. Turn on:
   `Required`

### Question 3

1. Add another question.
2. Type:
   `Action`
3. Set the question type to:
   `Multiple choice`
4. Add these choices:
   - `Sign in`
   - `Sign out`
5. Turn on:
   `Required`

## Part 3: Check the settings

1. Click `Settings`.
2. Keep the form simple.
3. Do not add extra questions unless Simon asks for them.
4. Leave the automatic Google timestamp on. This is important because Google will record the date and time for every response.

Important:
- Do not create a separate date question.
- Do not create separate forms for sign in and sign out.
- Do not make a new form every day.

## Part 4: Link the form to a spreadsheet

1. Click the `Responses` tab.
2. Click the green `Create spreadsheet` button.
3. Choose:
   `Create a new spreadsheet`
4. Name it:
   `Computer Sign In Out Register`
5. Click `Create`

This sheet will automatically record:
- date and time
- student name
- computer number
- sign in or sign out

## Part 5: Test the form

1. Click `Preview` to open the form.
2. Enter a test name.
3. Choose a computer.
4. Select `Sign in`.
5. Submit the form.
6. Open the linked spreadsheet.
7. Check that the response appears correctly.
8. Do a second test using `Sign out`.

If both entries appear in the spreadsheet, the system is working.

## Part 6: Make it easy for students to use

After the form is working:

1. Copy the form link.
2. Open each classroom computer.
3. Add the link to the browser bookmarks bar.
4. Rename the bookmark:
   `Computer Sign In / Out`

If needed, also:
- place the link on the desktop
- create a QR code for the room
- print a short instruction sign near the computers

## Part 7: Daily student routine

Students should do this every day:

### At the start
- open the `Computer Sign In / Out` form
- type their name
- choose their computer
- select `Sign in`
- click `Submit`

### At the end
- open the same form again
- type their name
- choose their computer
- select `Sign out`
- click `Submit`

## Part 8: What Simon wants kept simple

Keep this system:
- simple
- repeatable
- easy

That means:
- one form only
- three questions only
- one spreadsheet only
- same link every day

Do not add:
- email collection
- extra comments boxes
- class name fields
- separate morning and afternoon forms
- separate sign-in and sign-out forms

## Part 9: Final check

Before finishing, make sure:
- the form title is correct
- the 3 questions are correct
- all questions are required
- all computer numbers are listed
- the spreadsheet is linked
- a test sign in worked
- a test sign out worked
- the form link is easy to find on classroom computers

## Exact question wording to use

Use this exact wording:

- `Name`
- `Computer`
- `Action`

Use these exact action choices:

- `Sign in`
- `Sign out`

## If something goes wrong

Check these things first:
- Are you signed into the correct Google account?
- Did you link the form to a spreadsheet?
- Did you turn on `Required` for all questions?
- Did you test both sign in and sign out?
- Did you use one form instead of creating more than one?

## Completion note

When the setup is finished, tell Simon:

`The Google Form and linked spreadsheet are set up, tested, and ready to use each day.`
