Finding a Creative Commons Image

Use Google Image Search to find a Grand Canyon image, filter by Creative Commons licences, check the original source, and write a simple image credit.

Testing draft. Google helps us find images, but Google does not own the images.

20-30 minutes

Purpose

Practise choosing an image that is safer to reuse in a document, slideshow, poster, or class activity.

Search term

Grand Canyon

Digital focus

Search skills, image licensing, copyright awareness, and safe reuse of online content.
