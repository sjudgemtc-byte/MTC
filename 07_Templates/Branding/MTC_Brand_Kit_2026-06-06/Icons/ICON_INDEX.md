# MTC Icon Index

Saved: 2026-06-06

Purpose: quick descriptions to help choose the right MTC icon for slides, handouts, worksheets, and teaching collateral.

## How to use this index

- Use the filename to find the PNG in this folder.
- Use the description for a quick visual check.
- Use the "Good for" note when choosing icons for a topic, module, or slide.
- Contact sheets are saved in `_index/` for quick browsing.
