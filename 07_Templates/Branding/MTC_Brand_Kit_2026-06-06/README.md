# MTC Brand Kit

Saved: 2026-06-06

This folder stores the MTC logo files for use in teaching collateral and other reusable templates.

Files included:
- `MTC_Logo_RGB.png` - standard full-colour screen logo
- `MTC_Logo_Black_RGB.png` - black screen logo for light backgrounds
- `MTC_Logo_White_RGB.png` - white screen logo for dark backgrounds
- `MTC_Logo_Rev_RGB.png` - reversed screen logo variant
- `MTC_Logo_CMYK.png` - standard full-colour print logo
- `MTC_Logo_Black_CMYK.png` - black print logo for light backgrounds
- `MTC_Logo_White_CMYK.png` - white print logo for dark backgrounds
- `MTC_Logo_Rev_CMYK.png` - reversed print logo variant
- `source-archive.zip` - original screen-logo archive
- `source-archive-print-logos.zip` - original print-logo archive

Icons:
- `Icons/` - icon library folder for teaching collateral and slides
- `Icons/source-archive-icons.zip` - original icons archive
- `Icons/` currently includes 72 extracted icon PNG files plus the archive copy
- `Icons/ICON_INDEX.md` - plain-English descriptions and suggested uses for the icon set
- `Icons/_index/` - generated contact sheets for quick visual browsing

Brand guidance:
- `Brand_Guidelines/` - reference documents for tone, look, and brand use
- `Brand_Guidelines/MTC FutureReady_LivingOurBrand_July2025.pdf` - saved brand guidance PDF
- `BRANDING_KIT_INSTRUCTIONS.md` - practical instructions for creating MTC teaching collateral
- `BRANDING_KIT_INSTRUCTIONS.txt` - plain text version for easy reuse in the MTC project
